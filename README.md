This database design is made for small thrift and handmade product store. It basically stores the information about customers, products, orders, payments, and shipping. 
Each customer can place orders, and each order can contain products. Product details such as color, size, quantity, and price are stored properly.
Orders are able to track payment and also giving shipping status.
